About the page morphs

This site links dist/components/view-transitions.css — the opt-in cross-document view-transitions module (T-89). The wordmark, the hero panorama, and the featured entry's artwork carry --wel-vt names, so they morph between pages in Chrome and Safari over http. Firefox falls back to instant navigation, and so does file:// (each file is its own origin — serve the folder to see the morphs). Under prefers-reduced-motion the module switches itself off entirely.
